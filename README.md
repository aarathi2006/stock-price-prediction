# stock-price-prediction
This project focuses on predicting stock prices using machine learning and deep learning techniques based on historical market data. It leverages time-series forecasting models to analyze stock price trends and generate accurate future price predictions.
# Technologies Used

Python

NumPy

Pandas

Matplotlib

Scikit-learn

TensorFlow / Keras

Jupyter Notebook

The project uses historical stock market data containing features such as:

Open price

High price

Low price

Close price

Volume

Adjusted Close
